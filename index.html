Specific problems we are currently working on include:
  • Next Generation Sequencing: Developing analysis tools for various applications including RNA-Seq, single cell expression and WGBS.
  • Quantitative Epigenetics: Analyzing high throughput DNA methylation data.
  • Developing Bioconductor packages: including minfi, bumphunter, and alpine.
  • Gene Expression Barcode: Single sample classification based on calling expressed genes or transcript.
  • RMA, fRMA, GCRMA: Preprocessing algorithms for Affymetrix arrays.
The Publications Section provides links to papers on all these topics. The Software and Data Section provides Bioconductor tools.
